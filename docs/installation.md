# Installation Guide

## Node.js
OpenInfra Logger is designed to have zero external production dependencies, ensuring maximum security and minimum footprint.

### Using NPM
```bash
npm install openinfra-logger
```

### Using Yarn
```bash
yarn add openinfra-logger
```

## Python (Coming Soon)
The Python implementation is currently under development. It will be available via PyPI using `pip install openinfra-logger`.
